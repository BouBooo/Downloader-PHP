# Downloader-PHP

## A propos
  - Environnement: Windows 
  - Langage : PHP
  - Framework :  Symfony 4

## Fonctionnalités
#### Télécharger des musiques venant de :

 - Youtube
 - Soundcloud
 - Bandcamp
 
#### Possibilité de téléchargement :

 - Musique
 - Playlist
 - Album
 
 
 
 ## Installation
 
 ```bash
composer install
 ```
 
  ```bash
php bin/console make:migrations
 ```
 
  ```bash
php bin/console doctrine:migrations:migrate
 ```
